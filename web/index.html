Tutorial: from CAD to results

Walk through a complete analysis end to end — from a STEP file to a colour-mapped result — in about a minute in the browser. Every figure below is a real screenshot of the solver, captured automatically by driving the app through this exact workflow (see web/tests/tutorial-capture.spec.ts), so it always matches the current UI.

Import the geometry

Launch the solver and drag in a STEP file — here a hollow tube. .step and .stp are both accepted from the import card.

See the CAD surface

KoFEM tessellates the boundary with OpenCASCADE, so the real shape appears in the viewport. Pick a material — steel is the default — and set the mesh element-size bounds.

Mesh the volume

Click Mesh STEP volume to fill the solid with tetrahedra using Netgen's OpenCASCADE integration. The status bar reports the node and element counts once it is solver-ready.

Constrain and load

In Constraints, fix one face (held in all three directions, or just one — a single-DOF roller for a symmetry plane is supported) and add a load on another. The fixed face and the load arrow are highlighted on the model.

Solve and read the results

Run the MFEM linear-elastic solve. Displacements and von Mises stress come back in seconds and colour the model; switch the result field and read the min/max in the summary panel.
